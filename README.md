#  Chat UI Interface (Next.js)

## 📌 Project Description
This project is a Chat UI Interface built using Next.js with the App Router.  
It simulates a real-time chatbot experience with a mock backend API.  
The application demonstrates frontend-backend communication, state management, and responsive UI design.

---

##  Features
- Chat interface with user & bot messages
- Real-time message updates
- Typing indicator (Bot is typing...)
- Auto-scroll to latest message
- Timestamp for each message
- Dark mode toggle 
- Clear chat button 
- Responsive design (mobile + desktop)
- Mock API with delay & random responses

---

##  Tech Stack
- Next.js (App Router)
- React (useState, useEffect)
- TypeScript
- Tailwind CSS
